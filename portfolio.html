Trend.ng

Trend.ng is a way to visualize the top 10 trends on Twitter using pictures found on Instagram. Twitter's top 10 trends are always changing and Trend.ng can be a fun way to keep up with current topics. Trend.ng was built using Ruby on Rails/jQuery and implements the Twitter and Instagram APIs.



















FriendIndeed

FriendIndeed is a collaborative effort to make the world a more generous place. We attempt to do this by using social media platforms such as Facebook to connect users. Users will be able to log the good deeds and tag their friends that they helped. The simple to use interface will allow users to easily "pay it forward". We want users to be able to see the growth of their single deed in a form of a table and graph. FriendIndeed was built using Ruby on Rails and D3 JavaScript framework.
